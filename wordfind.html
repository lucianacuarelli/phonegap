Word Game Summary

The jQuery Word Game project is a plugin that creates word grids for use in word games.

5x5 Grid

Please read the word game documentation for instructions on how to use the plugin.

Options

classmodifier
Used to apply class names to the elements created by the plugin. You only need to specify a class modifier if you have a name-clash.
gridwidth
The number of columns to display.
gridheight
The number of rows to display.
allowable
A comma separated list of allowable values.
